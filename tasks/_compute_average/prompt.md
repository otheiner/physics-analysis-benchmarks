## Problem definition

The provided file contains numbers. You must:

1. List all numbers you found, one per line
2. Compute their average
3. State your final answer explicitly
_______________________________________
## Output format

Structure your response exactly as follows:

NUMBERS FOUND:
<number 1>
<number 2>
...

AVERAGE:
<your computed average>
________________________________________

## Files provided

The input data is one text file with numbers that should be used to compute the average.

- `input_numbers.txt` — File with numbers used to compute average
